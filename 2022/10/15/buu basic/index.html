简单的ssh连接进入远程端

LAMP 是指Linux（操作系统）+ Apache （HTTP 服务器）+ MySQL（数据库）和 PHP（网络编程语言），一般用来建立 web 应用平台。

连接操作

ssh连接，输入用户名+@+地址。参数根据需求添加，比如这里指定了连接端口

cd / 跳转到根目录，之后ls打印出当前目录的列表进行查看找到flag

进入后cd

参数：

  • -p：指定远程服务器上的端口；

  • -F：指定ssh指令的配置文件；

  • -o：指定配置选项；

  • -g：允许远程主机连接主机的转发端口；

  • -i：指定身份文件；

  • -l：指定连接远程服务器登录用户名；

  • -b：使用本机指定地址作为对应连接的源ip地址；

  • -1：强制使用ssh协议版本1；

  • -2：强制使用ssh协议版本2；

  • -4：强制使用IPv4地址；

  • -6：强制使用IPv6地址；

  • -A：开启认证代理连接转发功能；

  • -a：关闭认证代理连接转发功能；

  • -C：请求压缩所有数据；

  • -f：后台执行ssh指令；

  • -N：不执行远程指令；

  • -q：静默模式；

  • -X：开启X11转发功能；

  • -x：关闭X11转发功能；

  • -y：开启信任X11转发功能。

先学习一下反序列化漏洞

序列化就是把一个对象变成可以传输的字符串。

序列化实际是为了传输的方便,以整个对象为单位进行传输, 而序列化一个对象将会保存对象的所有变量，但是不会保存对象的方法，只会保存类的名字。如果了解底层的同学可以知道,类中的方法本就不在类中。

在php中,使用函数serialize()来返回一个包含字节流的字符串来表示

比如:

序列化的结果是:

O:1:"S":1:{s:4:"test";s:2:"sd";}

再比如本题

结果：

O:3:"BUU":2:{s:7:"correct";s:0:"";s:5:"input";R:2;}

O:3表示了类的名字长度为3；”BUU”:2表示类名为BUU，里面有两个变量。其他类似

主要表示方法就是从大的到小的，先是较为泛的变量类型以及变量名长度，之后是变量名以及其内容长度大小

PHP 对不同类型的数据用不同的字母进行标示

  • a - array
  • b - boolean
  • d - double
  • i - integer
  • o - common object
  • r - reference
  • s - string
  • C - custom object
  • O - class
  • N - null
  • R - pointer reference
  • U - unicode string

反序列化没问题，但是反序列函数的参数由用户可控就有问题了，完全可以编写函数序列化后利用反序列化调用

关于不同数字md5后值相同

1.弱等于

不需要寻找MD5碰撞的例子，PHP有个隐式转换的缺陷。

md5弱类型比较，为0e开头的会被识别为科学计数法，结果均为0

PHP在处理哈希字符串时，会利用”!=”或””来对哈希值进行比较，它把每一个以”0E”开头的哈希值都解释为0，所以如果两个不同的密码经过哈希以后，其哈希值都是以”0E”开头的，那么PHP将会认为他们相同，都是0。

构造例子
QNKCDZO
240610708

当然，相对安全的比较方法是三个=，也就是“===”。（强类型，数组绕）
